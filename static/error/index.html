Error: Cannot find Mongo Collection

This will need to be corrected in your App Studio by:

Upgrade the collection for this tool

Click into this app's data browser (for the current environment), then click 'upgrade database' to upgrade the collection to dedicated collection and then click 'Upgrade now'.

When doing this the Studio will require a re-deploy of your app, so click the 'deploy' button in the prompt that shows up after upgrading.

Re-deploy the monitoring tool

At this point, some new node modules are required by the monitoring tool, so a 'clean deploy' is required to have it install the new requirements. To do this, put a check mark in 'clean stage' then click 'deploy cloud app'

Confirmation

Once the deploy is finished, come back to this page and refresh it, and this message should go away.
